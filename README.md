# AI Social Engineering Lab

> **Educational Purpose Only** - This lab demonstrates AI-powered social engineering attacks for cybersecurity education and penetration testing training.

## Overview

This lab demonstrates how attackers leverage AI to create convincing phishing campaigns at scale. The toolkit includes:

- **PhishGPT**: AI-powered phishing email generation
- **GoPhish Integration**: Automated campaign management
- **Real-time Analytics**: Track attack effectiveness
